# 🍳 Learning Claude Code — The Kitchen

An interactive learning platform teaching Claude Code to non-developers through a kitchen metaphor.

## Levels

| Level | Name | Description |
|-------|------|-------------|
| 1 | Setup the Kitchen | Install tools, learn VS Code, first missions |
| 2 | Open the Restaurant | Build and publish a real webpage |
| 3 | Chef's Night Off | Build and ship a browser game |
| 4 | The Kitchen Brigade | Real GitHub collaboration |
| 5 | Your Kitchen, Your Menu | Build something for your real life |

## Setup for GitHub Pages

1. Fork or clone this repo
2. Go to Settings → Pages → Source: Deploy from branch → main
3. Your site will be live at `https://yourusername.github.io/repo-name`

## Analytics Setup

See `analytics.js` — replace the endpoint URL with your Google Apps Script deployment URL.
Admin dashboard available at `/admin.html`.

## Tech

Single HTML file. No framework. No build step. Works everywhere.
